Implementations of ASCII Tables using 7-bit ASCII as well as UTF-8 characters for borders. The implementations come in different versions (currently V1 and V2) with various degrees of flexibility and customization. The usage of each version is explained in the related package description (package v1 and package v2).

This blog post explains how to get UTF-8 support in the whole tool chain. Updates will be at the SKB Wiki on UTF-8.
